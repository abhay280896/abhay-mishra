# Abhay Mishra Portfolio

A modern single-page portfolio website built with React and Vite using the resume content you provided.

## Tech Stack

- React
- Vite
- Custom responsive CSS

## Run locally

1. Install dependencies:

```bash
npm install
```

2. Start the development server:

```bash
npm run dev
```

3. Build for production:

```bash
npm run build
```

## What is included

- Hero section with professional summary
- Skills grouped by category
- Experience timeline with project highlights
- Education section
- Responsive layout for desktop and mobile

## Customization ideas

You can easily extend this portfolio by adding:

- Your email, LinkedIn, and GitHub links
- Project case studies
- A downloadable resume button
- Testimonials or certifications
